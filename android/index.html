Política de Privacidade - ScanNut

Última atualização: 23 de Dezembro de 2025

1. Coleta de Dados

O ScanNut coleta informações fornecidas pelo usuário para a gestão da saúde animal, incluindo nome do pet, raça, histórico de saúde e nutrição.

2. Câmera e Microfone

O acesso à câmera é utilizado exclusivamente para capturar fotos de feridas para análise e galeria. O microfone é utilizado apenas para a funcionalidade de "Voz para Texto" nas observações. Nenhuma mídia é coletada sem a ação explícita do usuário.

3. Inteligência Artificial

As imagens de feridas são processadas por IA para fins informativos. Estes dados são tratados de forma anônima para melhorar a precisão do modelo.

4. Armazenamento e Exclusão

Os dados são armazenados de forma segura no Firebase (Google Cloud). O usuário tem o direito de excluir sua conta e todos os dados associados a qualquer momento através das configurações do app.

5. Isenção de Responsabilidade

O ScanNut não substitui o diagnóstico de um médico veterinário.

Privacy Policy - ScanNut

Last Updated: December 23, 2025

1. Data Collection

ScanNut collects information provided by the user for pet health management, including pet name, breed, health history, and nutrition.

2. Camera and Microphone

Camera access is used exclusively for capturing wound photos for analysis and gallery. The microphone is used only for the "Voice-to-Text" feature in notes. No media is collected without explicit user action.

3. Artificial Intelligence

Wound images are processed by AI for informational purposes. This data is handled anonymously to improve model accuracy.

4. Storage and Deletion

Data is securely stored in Firebase (Google Cloud). Users have the right to delete their account and all associated data at any time via the app settings.

5. Disclaimer

ScanNut is not a substitute for professional veterinary diagnosis.
