Resumo de Privacidade: O ScanNut adota a política de "Dados Locais Primeiros". Todas as informações de saúde, nutrição e biohacking são armazenadas exclusivamente no seu dispositivo utilizando a tecnologia Hive. Não realizamos coleta centralizada de dados em nuvem.

1. Coleta e Tratamento de Dados

O ScanNut processa informações para fornecer funcionalidades de Biohacking, Botânica e Gestão Pet. Os dados processados incluem:

  • Dados de Saúde Humana: Peso, altura e registros de treino para cálculos de performance.
  • Dados de Saúde Animal: Prontuário clínico, fotos de lesões e histórico de vacinação.
  • Dados Biofílicos: Fotos de plantas e diagnósticos de ecossistemas.

2. Uso de Sensores e Permissões

Em conformidade com as políticas da Google Play, declaramos o uso de:

  • Câmera: Utilizada exclusivamente para a captura de imagens de alimentos, plantas e pele de pets para processamento via Inteligência Artificial.
  • Microfone: Utilizado para a funcionalidade de transcrição de voz para texto (diário clínico). O áudio é processado em tempo real e não é armazenado como arquivo sonoro.

3. Inteligência Artificial e Segurança

As análises visuais do ScanNut são realizadas por modelos de IA. As imagens são enviadas de forma transiente para processamento e o resultado é devolvido ao armazenamento local do usuário. Nenhuma imagem é utilizada para identificação facial ou fins publicitários.

4. Direitos do Titular (LGPD/GDPR)

Visto que o armazenamento é local, o usuário possui controle total:

  • Exclusão: A exclusão dos dados é feita instantaneamente ao desinstalar o app ou limpar o cache do sistema.
  • Portabilidade: O usuário pode exportar seus dados a qualquer momento via relatórios PDF gerados pelo app.

5. Isenção de Responsabilidade (Termos de Uso)

O ScanNut é uma ferramenta de apoio informativo. NÃO SUBSTITUI DIAGNÓSTICOS MÉDICOS OU VETERINÁRIOS. A Multiverso Digital não se responsabiliza por decisões tomadas com base nas sugestões de biohacking, planos de recuperação botânica ou análises de IA. Consulte sempre um profissional qualificado.
